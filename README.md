# 🎮 Word Guessing Game (Java)

A simple **Word Guessing Game** built using **Java**.
The game selects a random English word and the player must guess it **letter by letter** within **5 attempts**.

## 🕹️ Game Features

* Random English word selection
* Fill-in-the-blank word display (`_ _ _ _`)
* Player guesses letters
* Correct letters reveal their position in the word
* Score system
* 5 attempt limit
* Duplicate letter protection
* Play Again option

---

## ⭐ Score System

| Action         | Score |
| -------------- | ----- |
| Correct Letter | +10   |
| Wrong Letter   | -2    |
